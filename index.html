Dict.get() with Mutable Defaults

Python

F-String Debugging

Python 3.8+ lets you debug with f-strings using the = suffix.

x = 42
name = "debug"

# Instead of this:
print(f"x: {x}, name: {name}")

# Do this:
print(f"{x=}, {name=}")
# Outputs: x=42, name='debug'

Rust

Use .unwrap_or_default()

Cleaner than unwrap_or() when you just want the type's default value.

// Instead of:
let value = option.unwrap_or(0);
let text = option.unwrap_or(String::new());

// Just do:
let value = option.unwrap_or_default();
let text = option.unwrap_or_default();
